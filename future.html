My Projects

Project 1 (ARDUINO PROJECT)

Arduino-Based Wheelchair Fall Detection System Designed and developed a smart fall detection system for wheelchair users using Arduino (Arduino Uno). The system uses an accelerometer and gyroscope sensor (e.g., MPU6050) to monitor sudden tilt, impact, or abnormal movement. When a fall is detected, the device automatically triggers an alert through a buzzer and sends emergency notifications via a GSM module (e.g., SIM800L) to predefined contacts. This project enhances safety for elderly and disabled individuals by providing real-time monitoring and quick emergency response. It demonstrates skills in embedded systems, sensor integration, IoT communication, and real-time data processing.

Project 2 (RASPBERRY PI PROJECT)

Raspberry Pi Based Underwater Acoustic Sensor Network Designed and implemented a prototype underwater communication system using Raspberry Pi to enable acoustic data transmission between submerged sensor nodes. Developed signal processing algorithms for underwater sound communication and tested real-time data exchange for environmental monitoring applications such as water quality analysis and marine surveillance.
